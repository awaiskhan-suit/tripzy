# tripzy
Online ticket booking app having two modules one is for Admin and another is for customer.
Customer can create account having name,phone number, email, password
Then this customer can also login
First of all Admin can add bus with thier detail and also see the bus detail,update the bus record, delete the bus record
Customer can book the ticket of bus with their name, phone, and select the payment method 
Then admin can view thier booking history and change their ticket status i-e pending, complete, finish, canclled, 
When admin change the status of ticket so the customer can see the status of ticket also download the ticket information in pdf form
Admin can see the number of customer, no of tickets that is book , pending tickets, today sales,monthly sales, year sales and also see the payment method sales i-e jazzcash,easypaisa,by bank
